# **Что это?**

Прикольное учебное приложение для рисования с использованием веб-камеры. Для него необходима установка OpenCV (с указанием пути в CMakeLists.txt и run.ps1).

Работает так - https://t.me/dkriskovets/363


# **Как использовать?**
  
0. Если нет желания настроить какие-то цвета кроме рыжего и желтого - начать с пункта 5.
1. Собрать и запустить color_detector.cpp из archive
2. Настроить ползунками поиск цвета по колпачку маркера или по какому-то другому ИРЛ объекту.
3. Скопировать значения настройки цвета из вывода в консоль.
4. Открыть main.cpp, вставить настройки цвета в вектор my_irl_marker_colors и задать ему цвет отображения в векторе my_digital_colors.
5. Запустить run.ps1 из папки проекта
